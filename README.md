# Vue + Spring Boot 项目名称

项目简介：个人博客

## 技术栈
- 前端：Vue.js, Vue Router, Axios
- 后端：Spring Boot, Spring Security, MyBatis/MySQL

## 环境要求
- JDK 17+
- Node.js 14+
- MySQL 8.0+

## 运行步骤

### 后端
1. 进入backend目录：`cd backend`
2. 配置数据库信息：`src/main/resources/application.properties`
3. 启动项目：`mvn spring-boot:run`

### 前端
1. 进入frontend目录：`cd frontend`
2. 安装依赖：`npm install`
3. 启动项目：`npm run serve`

## 项目结构
- `/backend`：Spring Boot后端代码
- `/frontend`：Vue前端代码
